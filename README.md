# Customer-behavior-analysis
This project focuses on analyzing customer purchasing behavior to uncover insights that can help businesses make data-driven decisions. The analysis was performed using SQL for querying, Python (Jupyter Notebook) for data processing, and Power BI for interactive dashboard visualization.

📂 Dataset
The dataset contains customer transaction and demographic information, including:
1)Customer ID, Age, Gender
2)Product details (Item, Category, Size, Color)
3)Purchase Amount and Review Ratings
4)Subscription status, Discounts, Shipping type
5)Previous purchases and buying frequency

🛠️ Tools & Technologies
1)Python (Jupyter Notebook) – Data loading, cleaning, and EDA
2)Pandas, NumPy, Matplotlib/Seaborn – Data analysis and visualization
3)Microsoft SQL Server – Data storage and querying
4)Power BI – Interactive dashboard creation
5)Gamma – Presentation (PPT) generation

🔄 Project Workflow
1. Data Loading
Imported dataset into Jupyter Notebook using Pandas
2. Data Cleaning
Handled missing values
Removed duplicates
Standardized column formats
3. Exploratory Data Analysis (EDA)
Analyzed customer demographics
Identified trends in purchase behavior
Visualized key patterns using charts
4. SQL Analysis (Microsoft SQL Server)
Loaded cleaned data into SQL Server
Performed queries for:
Revenue analysis
Customer segmentation
Product performance
Discount and subscription impact
5. Dashboard Creation (Power BI)
Built interactive dashboard with filters and KPIs
Visualized revenue trends, top products, and customer segments
6. Reporting & Presentation
Summarized insights into a structured report
Created presentation slides using Gamma

📊 Dashboard Highlights
Revenue and average purchase KPIs
Customer segmentation (New, Returning, Loyal)
Top-performing products and categories
Gender and age-based analysis
Impact of discounts and subscriptions

📈 Results & Insights
Subscribed customers show higher average spending
Discounts significantly influence purchasing behavior
Loyal customers contribute the highest revenue share
Certain categories consistently outperform others

▶️ How to Run
Clone the repository
Open Jupyter Notebook and run the analysis file
Load data into Microsoft SQL Server
Execute SQL queries from the /sql folder
Open Power BI file to explore the dashboard

🚀 Conclusion
This project demonstrates an end-to-end data analytics workflow, transforming raw data into actionable insights using Python, SQL, and Power BI.
